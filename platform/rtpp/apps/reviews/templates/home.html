Get started with Data Streams

Alot of data streams are available to the public yet it is difficult getting started. The infrastructure needed for such a setup slows down people who are just interested in a subset of the public data.

Our service allows you to easily access data streams and filter out the bits you want to take action on. You write a small user defined script to filter and transform the data you need and we will call your API endpoints and provide a query API to access those events.

This takes the burden from infrastructure of dealing with high volume streams and let's you work with the data you are actually interested in.
